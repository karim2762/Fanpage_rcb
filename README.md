## ✨ Overview

This project is a **high-end interactive fan website** dedicated to **Royal Challengers Bangalore (RCB)**.

Designed with modern UI/UX principles, smooth animations, and immersive visual effects, it delivers a premium browsing experience without requiring any frameworks or external build tools.

---

# 🚀 Features

## 🎬 Immersive Experience
- 🔥 Animated loading screen
- 🌌 Dynamic particle background
- 💫 Smooth transitions & reveal animations
- 🎯 Responsive layout for desktop and mobile

## 🏏 RCB Fan Hub
- 👑 Hero section with featured player
- 📅 Match fixtures & countdown timer
- 👥 Interactive squad showcase
- 📊 Team statistics dashboard
- 📰 Latest news section
- ❤️ Fan engagement zone
- ⭐ Iconic RCB moments gallery

## 🎨 Premium UI
- Glassmorphism effects
- Modern gradients
- Red & Gold RCB-inspired theme
- Hover interactions
- Scroll animations
- Custom cards and modal popups
